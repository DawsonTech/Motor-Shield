# Motor-Shield
Shield for Arduino to drive DC-Motors, Solenoids or Relays.

* Compatible with [Arduino Motor-Shield](https://www.arduino.cc/en/Main/ArduinoMotorShieldR3)
* ST L298 Dual Full-Bridge Driver (2A per channel or 4A max)
